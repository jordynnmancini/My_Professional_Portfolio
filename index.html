Watchaholic - a Movie Search App

This was done as a group project, and is built with HTML, CSS, Bootstrap, Javascript, JQuery, and includes Third-Party API Sources (WATCHMODE/OMDB/New York Times). This allows the user to search for the movie or t.v show title of their choice, and presents them with imporatant information about it, a link to a NYT Review (if one exisits), and shows a list of streaming services and other websites that are showing this title.
